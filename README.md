# 🗺️ Редактор исторических карт ВОВ

Настольное приложение для отрисовки линий фронта Великой Отечественной войны.
Построено на **PyQt6 + Leaflet.js**, работает на Windows 10/11.

---

## ⚙️ Установка

### 1. Требования
- Python 3.10–3.12 (64-bit)
- Windows 10 / 11

### 2. Установка зависимостей
```bash
pip install PyQt6 PyQt6-WebEngine
```

> Если `PyQt6-WebEngine` не ставится — попробуйте:
> ```bash
> pip install PyQt6-WebEngine --pre
> ```

### 3. Запуск
```bash
python main.py
```

---

## 🔑 Получение API-ключа Яндекс Карт

1. Откройте **[developer.tech.yandex.ru](https://developer.tech.yandex.ru)**
2. Войдите в Яндекс-аккаунт (или зарегистрируйтесь)
3. Нажмите **«Подключить API»**
4. Выберите **«JavaScript API и HTTP Геокодер»**
5. Заполните форму (название проекта, URL — можно указать `localhost`)
6. Скопируйте полученный ключ

При первом запуске приложение автоматически откроет диалог ввода ключа.
Ключ сохраняется в `config.json`.

> **Без ключа** приложение работает с картами **OpenStreetMap** (бесплатно, без регистрации).

## 🖌️ Инструменты

### Кисть (`B`)
- Зажмите ЛКМ и рисуйте линию фронта свободной рукой
- Настройте цвет и толщину в нижней панели

### Ластик (`E`)
- Зажмите ЛКМ и водите — стирает линии на активном слое
- `Shift + ЛКМ` — стирает на **всех** слоях
- Регулируйте размер ластика слайдером

### Текст (`T`)
- Кликните на карту → введите текст в диалоге
- Двойной клик по тексту на карте — редактировать
- Настройки: размер, цвет, жирный/курсив

### Выбор (`V`)
- Кликните объект для выделения (синяя подсветка)
- `Delete` — удалить выбранный объект

### Импорт (`🖼`)
- Открывает диалог выбора JPG/PNG
- Изображение помещается в центр текущего вида
- Настройка прозрачности через слайдер снизу

## 🌐 Интернет

Приложение требует интернет-соединение **только для загрузки тайлов карты**.
Все данные проекта хранятся локально.

---

## 🐞 Решение проблем

| Проблема | Решение |
|---|---|
| Белый экран вместо карты | Проверьте интернет-соединение |
| Ошибка импорта PyQt6-WebEngine | `pip install --upgrade PyQt6-WebEngine` |
| Яндекс-тайлы не загружаются | Проверьте API-ключ через кнопку 🔑 |
| Приложение не запускается | Убедитесь, что Python 64-bit |
